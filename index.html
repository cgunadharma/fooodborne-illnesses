Introduction

The data set, Daily Temperatures of Major Cities, was found on Kaggle and was collected by a data scientist at H2O.ai, an open source software company in Silicon Valley. There are 1,048,575 observations in this data set with eight features: region, country, start, city, month, day, year, and average temperature. This data set will go a long way in determining where the highest changes in temperature occurred and will allow us to measure multiple climate change variables. We will be able to answer questions such as: Which states have experienced the highest temperature change over time? Based on current temperature patterns, which cities will be heavily impacted by climate change and global warming in the near future?
